## README - Quiz Game Using Phaser

---

### Description

This **Quiz** game is built using **Phaser 3**, where the player answers multiple-choice questions with a single correct answer. The goal is to score as many points as possible by providing correct answers, with additional points awarded for remaining time. The questions are dynamically loaded from an encrypted JSON file using **CryptoJS**.

---

### Features

- **Quiz Questions**: Answer single-choice questions.
- **Scoring System**: Players earn points for correct answers and additional points for remaining time.
- **Timer**: Limited time to answer each question.
- **Responsive Gameplay**: The game works on both desktop and mobile devices.
- **Dynamic Questions**: Questions are loaded from an encrypted JSON file, decrypted with **CryptoJS**.
